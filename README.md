# FODMAP–GI Frontend

**Live Demo:** https://fodmap-gi-frontend.vercel.app/

This repository contains the **frontend application** for the FODMAP–GI project.

The application allows users to:
- Register a profile
- Log meals and symptoms
- Check menu items that may be associated with GI symptoms

## Scope
This is a **frontend-only application** built to demonstrate UI design, routing, and user interaction.

No backend services are required to run or review this project.

## Tech Stack
- React
- TypeScript
- Tailwind CSS
- Vite
- React Router

## Deployment
- Hosted on Vercel
- SPA routing configured via `vercel.json`
